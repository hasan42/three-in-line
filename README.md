# ThreeInLine

[DEMO](http://ng-balls.irustam.ru/)

Игра "три в ряд" на Angular.

Стили в SCSS.

В мобильной версии:
- уменьшено поле;
- уменьшено количество цветов;
- убраны тени/glow эффект для оптимизации.
